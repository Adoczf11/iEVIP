# R code for iEVIP: A Dual-inspired Intelligent Platform for Smart Response Extracellular Vesicle Isolation and Single-Vesicle Profiling
This GitHub repository contains R scripts used for data visualization, multivariate analysis, and classification in this study, including Heatmap,PCA, PLS-DA, LDA, and  RF + LDA + NN (k = 5).

The file 20240614.csv contains data derived from cell-based experiments. Each row corresponds to an independent cell sample, and columns represent measured features used for subsequent multivariate analyses, including heatmap visualization, PCA, PLS-DA, LDA, and combined classification models.

## Description of individual scripts
### **Heatmap.R**
This script performs data normalization (if applicable) and generates heatmaps to visualize global expression or feature patterns across samples. It is used to explore overall sample similarity, clustering patterns, and potential group-specific trends.
---
### **PCA.R**
This script conducts **principal component analysis (PCA)** as an unsupervised dimensionality reduction method. PCA is used to visualize the overall structure of the dataset, identify dominant sources of variance, and detect potential outliers.

### **PLS-DA.R**
This script performs **partial least squares discriminant analysis (PLS-DA)** as a supervised multivariate classification approach. It is used to assess group separation and identify features contributing most strongly to class discrimination.

### **LDA_visualization.R**
This script applies **linear discriminant analysis (LDA)** as a linear and interpretable classification method. LDA serves as a baseline classifier for comparison with nonlinear models.

### **tSNE.R**
This script performs t-distributed stochastic neighbor embedding (t-SNE) as a nonlinear dimensionality reduction approach. It is used to visualize sample clustering patterns and group separation in a low-dimensional space.

### **ComBat_batch_correction.R**
This script performs batch-effect correction for features shared across different sample batches. It is used to reduce nonbiological inter-batch variation, visualize correction performance by principal component analysis (PCA), and generate batch-corrected data for subsequent classification analysis.

## **RF+NNET+LDA+Confusion_matrix+ROC+PR.R**
This script implements and compares multiple classification models, including Random Forest (RF)，Neural Network (NN) and Linear Discriminant Analysis (LDA). It is used to evaluate classification performance under different modeling assumptions and to assess model robustness.

## Reproducibility
All scripts were executed using **R version 4.3.0**. Where applicable, random seeds were set within the scripts to ensure reproducibility. Output figures and tables generated by the scripts correspond to those presented in the manuscript.

##Data availability (for manuscript)
The source data used to generate each figure are provided in files named according to the corresponding figures. The cell_EV dataset contains cell-derived EV data and was used for visualization and classification analysis using multiple machine-learning algorithms. The Batch1_EV and Batch2_EV datasets contain EV data from two independent batches of clinical samples. The features dataset contains the combined data from these two batches and was used for batch-effect correction. The Fig9G dataset contains the batch-corrected data and was used to validate the generalizability of the method.
